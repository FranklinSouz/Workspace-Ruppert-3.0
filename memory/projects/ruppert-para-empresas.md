# Projeto: "Ruppert para Empresas" — AI Agency

**Data da ideia:** 2026-03-27
**Status:** Conceito validado, aguardando execução

## Resumo do modelo
Instalar OpenClaw + agente personalizado para empresas terceiras.
- Starter: R$2.500 setup + R$500/mês
- Pro: R$5.000 setup + R$1.200/mês
- Empresa: R$10.000+ setup + R$2.500/mês

## ICP e Nichos prioritários
**Perfil do cliente ideal:** Empresário com operação rodando e receita real, mas tempo escasso. Já ouviu falar de IA, sente que tá ficando pra trás, não vai estudar, tem funcionário. Gatilho de compra = "tô perdendo dinheiro com isso".

1. Clínicas e consultórios (médico, odonto, psico, estética) — dor: "paciente mandou mensagem à noite e marcou na concorrência"
2. Imobiliárias pequenas / corretores — dor: "pago caro em tráfego e o lead esfria"
3. Escolas de idiomas / cursos presenciais — secretária sobrecarregada

**Evitar agora:** restaurantes (ticket baixo), e-commerce pequeno (complexidade alta), advogados/contadores (decisão lenta)

## Estrutura de workspace por cliente
```
SOUL.md → persona do agente
BUSINESS.md → serviços, horários, preços, políticas
FAQ.md → perguntas e respostas reais
RULES.md → o que pode/não pode fazer
memory/customers.md → histórico de clientes
memory/decisions.md → decisões do dono
```

## Ordem de instalação
1. Onboarding (entrevista 1h)
2. Criar workspace + copiar template do segmento
3. Personalizar SOUL + BUSINESS + FAQ
4. Conectar WhatsApp
5. Conectar integrações
6. Testar 20 conversas reais
7. Go live + monitorar 1ª semana
8. Handoff: Loom 10min pro dono
9. Revisão mensal
